Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide

ETDA-MCDL-Observation-GA

profile สำหรับ Observation resource ใช้บันทึกข้อมูลผลการตรวจร่างกายทั่วไป
ETDA-MCDL-Encounter

profile สำหรับ Encounter resource ใช้บันทึกข้อมูลการตรวจ
ETDA-MCDL-Organization-ICT-MoPH

profile สำหรับ Organization resource ใช้บันทึกข้อมูลของผู้ดูแล FHIR Document นี้ ได้แก่ ศูนย์เทคโนโลยีสารสนเทศ สำนักงานปลัดกระทรวงสาธารณสุข
ETDA-MCDL-Condition

profile สำหรับ Condition resource ใช้บันทึกผลการวินิจฉัย
ETDA-MCDL-Bundle

FHIR Document สำหรับใบอนุญาตขับรถ
ETDA-MCDL-QuestionnaireResponse

profile สำหรับ Practitioner resource ใช้บันทึกข้อมูลในใบรับรองแพทย์ในส่วนที่ผู้ป่วยกรอกเอง
ETDA-MCDL-Location

profile สำหรับ Location resource ใช้บันทึกข้อมูลสถานที่ตรวจ
ETDA-MCDL-Provenance-Practitioner

profile สำหรับ Provenance resource ใช้บันทึกข้อมูลการลงลายมือชื่อของแพทย์
ETDA-MCDL-Practitioner

profile สำหรับ Practitioner resource ใช้บันทึกข้อมูลแพทย์
ETDA-MCDL-Organization-Provider

profile สำหรับ Organization resource ใช้บันทึกข้อมูลสถานพยาบาล
ETDA-MCDL-Patient

profile สำหรับ Patient resource ใช้บันทึกข้อมูลผู้ป่วย
ETDA-MCDL-Provenance-Patient

profile สำหรับ Provenance resource ใช้บันทึกข้อมูลการลงลายมือชื่อของผู้ป่วย
ETDA-MCDL-Composition

profile สำหรับ Composition resource ซึ่งใช้สำหรับจัดโครงสร้างของเอกสารใน FHIR document (Bundle resource)
